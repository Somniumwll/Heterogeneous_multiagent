## Guided Collaboration in Heterogeneous LLM-Based Multi-Agent Systems via Entropy-Based Understanding Assessment and Experience Retrieval

A multi-agent collaboration framework evaluated on three benchmarks: **GSM8K** (mathematical reasoning), **CVRP** (combinatorial optimization), and **MBPP** (code generation). This system implements two collaboration paradigms and evaluates them under three experimental conditions using information-entropy-based adaptive guidance and RAG memory retrieval.
